## Story

- [ ] Modulos removidos do backend: whatsapp, clicks, campaigns, leads, inbox, orders, events, dashboard, ad-accounts, ai-insights, settings

- [ ] Paginas removidas do frontend: dashboard, campaigns, leads, inbox, orders, events, ad-accounts, ai-insights, settings, teste
- [ ] Componentes removidos: sidebar, whatsapp-qrcode, whatsapp-device-card, spend-revenue-chart
- [ ] Prisma schema limpo — remover TODOS os models antigos (Campaign, Click, Lead, Message, Order, Event, AdAccount, User antigo, etc.)

- [ ] package.json atualizado: name="brazacommerce", description atualizada
- [ ] Manter intacto: docs/prototypes/ (template v3.0), packages/shared/, estrutura de workspaces

- [ ] Pagina raiz (/) mostra texto simples "braza.commerce — em construcao"
- [ ] `npm run build` passa sem erros em api e web
