## Story

Como usuario, preciso configurar a integracao com Yampi (secret key) e Meta Pixel (pixel ID + access token) em uma pagina de configuracoes.

## Acceptance Criteria

- [ ] GET /settings — retorna configuracoes atuais
- [ ] PATCH /settings — atualiza yampiSecretKey, defaultPixelId, defaultAccessToken
- [ ] Frontend /settings com layout tracker.braza
- [ ] Secao Yampi: webhook URL (read-only, copiar) + campo secret key + botao salvar
- [ ] Secao Meta Pixel: campo pixel ID + campo access token + botao salvar
- [ ] Botao "Copiar" na webhook URL com feedback visual ("Copiado!")
- [ ] Status de conexao Yampi: dot verde live-pulse se ultimo webhook < 5min, cinza se > 5min ou nunca
- [ ] Settings criado automaticamente na primeira vez (upsert)
- [ ] Access token mascarado na UI (so mostra ultimos 4 chars)

## Dev Notes

- Settings e singleton (id = "default")
- Webhook URL gerada dinamicamente: `{API_BASE_URL}/api/webhooks/yampi`
- Status de conexao: verificar createdAt do ultimo Event no banco
- Upsert: se nao existe, cria; se existe, atualiza
