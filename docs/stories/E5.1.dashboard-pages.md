# Story E5.1 — Dashboard de paginas

> **Epic:** E5 - Dashboard
> **Status:** Done

## Story

Como usuario, quero ver um painel com todas as paginas criadas, seus status, e gerencia-las rapidamente.

## Acceptance Criteria

- [ ] Pagina /pages (home do app) — lista todas as paginas
- [ ] Cada card mostra: thumbnail (imagem position 1), titulo, status (badge colorido), data, link publico (se publicado)
- [ ] Badges: DRAFT (cinza), PUBLISHED (verde), ARCHIVED (vermelho)
- [ ] Acoes por pagina: Editar, Publicar/Despublicar, Copiar link, Duplicar, Deletar
- [ ] Botao "Nova pagina" no topo → vai para /pages/new
- [ ] Copiar link: copia URL publica para clipboard + feedback visual ("Copiado!")
- [ ] Duplicar: cria copia como DRAFT com novo slug
- [ ] Deletar: modal de confirmacao ("Tem certeza? Esta acao nao pode ser desfeita")
- [ ] Estado vazio: mensagem "Nenhuma pagina criada ainda" + botao "Criar primeira pagina"
- [ ] Ordenado por createdAt DESC (mais recente primeiro)
- [ ] Responsivo (2 colunas desktop, 1 coluna mobile)
- [ ] Sem login — acesso direto ao dashboard
