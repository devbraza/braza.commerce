# Story E2.3 — Regenerar campo especifico (texto ou imagem)

> **Epic:** E2 - Upload & AI
> **Status:** Deferred (SHOULD — implementar na v1.1)
> **Prioridade:** SHOULD

> **PRD:** FR-2.7, FR-2.8

## Story

Como usuario, quero poder pedir para a IA regenerar um campo especifico (texto ou imagem individual) sem regenerar tudo.

## Acceptance Criteria

- [ ] POST /pages/:id/regenerate — aceita { type: "copy", field: "description" } ou { type: "image", position: 3 }
- [ ] Para copy: regenera apenas o campo solicitado (name, brand, description, features, reviews, faq, miniReview)
- [ ] Para imagem: regenera apenas a imagem na posicao especificada (1-6)
- [ ] Copy regenerada faz merge no aiGeneratedContent (atualiza so o campo, nao replace total)
- [ ] Imagem regenerada substitui o arquivo no disco e atualiza PageImage
- [ ] Se usuario ja editou o campo (userEditedContent), o regenerate atualiza aiGeneratedContent mas NAO apaga userEditedContent

## Dev Notes
