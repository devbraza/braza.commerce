## Story

Como usuario, ao publicar minha pagina quero definir a URL personalizada (slug) para usar nos meus anuncios.

## Acceptance Criteria

- [ ] Campo de slug editavel no step de publicacao
- [ ] Prefixo fixo visivel: `braza.commerce/p/`
- [ ] Auto-slugify enquanto usuario digita (lowercase, hifens, sem caracteres especiais)
- [ ] Validacao em tempo real: verde = disponivel, vermelho = ja existe
- [ ] Endpoint GET /pages/check-slug/:slug retorna { available: true/false }
- [ ] Se usuario nao preencher, gera slug automatico (comportamento atual)
- [ ] Slug atualizado no PATCH /pages/:id antes de publicar
- [ ] Caracteres permitidos: a-z, 0-9, hifens
- [ ] Minimo 3 caracteres, maximo 60
