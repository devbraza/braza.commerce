## Story

Como usuario, quero publicar minha pagina e receber um link publico. Como visitante, quero acessar o link e ver a landing page completa.

## Acceptance Criteria

- [ ] PATCH /pages/:id/publish — muda status para PUBLISHED, salva publishedAt
- [ ] PATCH /pages/:id/unpublish — muda status para ARCHIVED
- [ ] GET /p/:slug — rota publica que renderiza e serve o HTML via render engine
- [ ] Pagina publica carrega < 2 segundos no mobile 4G
- [ ] Se slug invalido ou pagina nao PUBLISHED: retorna 404 limpo
- [ ] Imagens servidas de /uploads/ com cache headers (max-age 1 ano)
- [ ] Re-publicar (editar e publicar de novo) serve HTML atualizado
- [ ] Response HTML inclui Content-Type: text/html e cache de 5 min
- [ ] Pagina inclui todos os disclaimers de compliance (Meta + Google Ads)
- [ ] Todos os botoes "Comprar agora" apontam para o checkoutUrl correto

## Dev Notes

- MVP: gerar HTML on-the-fly a cada request (simples). Cache/pre-render na v2 se necessario
- A rota /p/:slug e a unica rota importante para SEO — servir HTML completo
- Headers: Cache-Control: public, max-age=300 (HTML), max-age=31536000 (imagens)
- Pagina 404 simples: fundo branco, texto "Pagina nao encontrada", sem template complexo
