## Story

Como usuario, preciso criar, editar, listar, duplicar e deletar minhas paginas de produto.

## Acceptance Criteria

- [ ] POST /pages — cria pagina nova (status: DRAFT), retorna id e slug
- [ ] GET /pages — lista todas as paginas (id, title, slug, status, createdAt, publishedAt, primeira imagem como thumbnail)
- [ ] GET /pages/:id — retorna pagina completa com imagens e conteudo (aiGeneratedContent + userEditedContent)
- [ ] PATCH /pages/:id — atualiza campos editaveis (title, price, originalPrice, checkoutUrl, userEditedContent)
- [ ] DELETE /pages/:id — deleta pagina, imagens do banco e arquivos do disco
- [ ] POST /pages/:id/duplicate — cria copia como DRAFT com novo slug
- [ ] Slug gerado automaticamente via slugify a partir do title (ou "pagina-" + random se sem titulo)
- [ ] Slug unico — se ja existe, adiciona sufixo numerico (-1, -2, etc.)
- [ ] Ordenacao por createdAt DESC na listagem
- [ ] Sem auth — todas as rotas sao publicas no MVP
