## Story

Como usuario, preciso criar campanhas vinculadas as minhas paginas para rastrear o funil de conversao.

## Acceptance Criteria

- [ ] Novos models no Prisma: Campaign, Click, Event, Settings (+ enums CampaignStatus, EventType)
- [ ] Relacao Page hasMany Campaigns
- [ ] Migracao aplicada com sucesso
- [ ] POST /campaigns — cria campanha (name, pageId, checkoutUrl, pixelId?, accessToken?)
- [ ] GET /campaigns — lista campanhas com mini stats (clicks, purchases, revenue)
- [ ] GET /campaigns/:id — detalhe com stats completo
- [ ] PATCH /campaigns/:id — editar (name, checkoutUrl, pixelId, accessToken)
- [ ] PATCH /campaigns/:id/pause — pausar
- [ ] PATCH /campaigns/:id/activate — ativar
- [ ] Frontend /campaigns — lista de campanhas com badges ATIVA/PAUSADA
- [ ] Frontend /campaigns/new — form criar campanha (dropdown pagina, checkout URL, pixel opcional)
- [ ] Layout tracker.braza em todas as telas

## Dev Notes

- Campaign vincula 1 pagina a 1 checkout URL + pixel opcional
- Mini stats na listagem: COUNT clicks, COUNT events WHERE type=PURCHASE, SUM value
- Badges: ATIVA (emerald), PAUSADA (zinc)
- Dropdown de paginas: so paginas PUBLISHED
