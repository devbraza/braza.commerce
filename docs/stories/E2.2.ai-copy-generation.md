# Story E2.2 — Geracao de copy com IA (Claude Vision)

> **Epic:** E2 - Upload & AI
> **Status:** Done
> **Prioridade:** MUST
> **Agente:** @dev (Dex)
> **Depende de:** E2.1 (precisa da foto de referencia)
> **PRD:** FR-2.3, secao 8 (prompt da IA)

## Story

Como usuario, apos fazer upload da foto de referencia, quero que a IA gere automaticamente todo o conteudo textual da minha landing page (nome, descricao, features, reviews, FAQ).

## Acceptance Criteria

- [ ] POST /pages/:id/generate-copy — envia foto de referencia para Claude API e gera conteudo
- [ ] Claude recebe a foto de referencia (base64) via vision
- [ ] Retorna JSON estruturado com: name, brand, description, features[4], reviews[4], faq[3], miniReview
- [ ] JSON salvo no campo aiGeneratedContent da Page
- [ ] Resposta em < 15 segundos
- [ ] Conteudo em portugues BR, tom persuasivo mas realista
- [ ] Reviews: 3 de 5 estrelas + 1 de 4 estrelas (credibilidade)
- [ ] Nomes dos reviewers sao brasileiros realistas
- [ ] FAQ: 3 perguntas relevantes ao tipo de produto
- [ ] Sem emojis nos textos
- [ ] Sem mencao a WhatsApp
- [ ] Erro 400 se pagina nao tem foto de referencia
- [ ] Erro 500 com mensagem amigavel se Anthropic API falhar
- [ ] Retry 1x se JSON parsing falhar

## Estrutura do JSON

```json
{
  "name": "string (max 60 chars)",
  "brand": "string (max 30 chars)",
  "description": "string (max 200 chars, 2-3 frases)",
  "features": [
    "string (max 60 chars, beneficio claro)",
    "string", "string", "string"
  ],
  "reviews": [
    { "stars": 5, "text": "max 150 chars", "author": "Nome X.", "verified": true },
    { "stars": 5, "text": "...", "author": "Nome Y.", "verified": true },
    { "stars": 5, "text": "...", "author": "Nome Z.", "verified": true },
    { "stars": 4, "text": "critica honesta mas positiva", "author": "Nome W.", "verified": true }
  ],
  "faq": [
    { "question": "O que esta incluso?", "answer": "max 200 chars" },
    { "question": "Prazo de entrega?", "answer": "..." },
    { "question": "Posso trocar?", "answer": "..." }
  ],
  "miniReview": {
    "initials": "XX",
    "stars": 5,
    "text": "max 100 chars, impactante",
    "author": "Nome · Compra verificada"
  }
}
```

## Dev Notes

- SDK: `@anthropic-ai/sdk`
- Model: `claude-sonnet-4-6` (vision + rapido + barato)
- ANTHROPIC_API_KEY no .env
- Custo: ~R$0.10 por geracao
- O prompt deve incluir regras do template v3.0 (tamanhos de texto, tom, estrutura)
- Se JSON.parse falhar, retry 1x com prompt de correcao
- Rate limit: 10 req/min
